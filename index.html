    Queen's University Tour Guide Application | Jan - Nov 2015

    I worked in a group of 5 to create an Android application for Queen’s University for the APSC 100 course. The application guides prospective students and parents around Queen’s University in the absence of a tour guide.

    My main role was lead developer/ designer and I programmed in Android Studio and Java. I created pen and paper designs, as shown on the left, of the application and the team decided to adopt the design for our application. The application is currently available in the Google Play Store, QEng Tour.

    QLife | CMPE 320 C++ Project | Sept - Dec 2016

    This project was created using C++ for the CMPE 320 course in an Agile team of 5 others. The calendar based application is designed to schedule any user-inputted tasks into the user’s schedule. A pie graph is used to help illustrate the users’ weekly activities. The app would recommend activities to balance their lifestyle based on their schedule.

    I made initial drawings of the application and the team decided to move forward with my design, shown on the left. I worked in the front-end team and helped design the structure of the application.

    Personal Website | May - June 2017

    I wanted to design a personal website to learn the key concepts of web design. Many hours were spent researching UX/UI design in order to create a great site. I drew an initial design of the site in my second year of University, however, much of the design has changed.

    Many comments are used to make the code as clear as possible. In addition, the documentation can be found on my Github with some details of the overall project. Credits to David Lao Photography for the home page photo. All other icons used belong to their rightful owner.

    deliverAI | Hack Western 4 (Hackathon) | November 2017

    DeliverAI is a simulation of a self-driving car delivery system. It delivers packages to the customer's address and features facial recognition as a method of authentication.

    I focused on the web application and hardware assembly for the project. I created original designs for the web application, implemented a real-time database using Firebase, and helped implement the Webcam API for sign up.

    Further documentation can be found in the DevPost. In addition the code can be found on Github.
